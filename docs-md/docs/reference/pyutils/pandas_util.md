Pandas utility functions.

#### tuples2DF

```python
def tuples2DF(tuples: list, columns) -> pd.DataFrame
```

Converts list of tuples into Pandas DataFrame for nice viewing.

**Arguments**:

- `tuples` - list of tuples.
  

**Returns**:

- `pd.DataFrame` - A DataFrame object.
