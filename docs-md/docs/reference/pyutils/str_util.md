String utility functions.

#### truncate\_string

```python
def truncate_string(s: str, max_length: int = 100) -> str
```

Truncate a string to a maximum length, adding &#x27;...[more n chars]&#x27; if truncated..

**Arguments**:

- `s` _str_ - The string to truncate.
- `max_length` _int_ - The maximum length of the string before truncation.
  

**Returns**:

- `str` - The truncated string.
